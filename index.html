Tor ERP Project

Tor is an ERP = Enterprise Resesource Management system. It enable a company to manage human resources, stocks and fixed assets. It assist accountants to handle computation, inputs and outputs for company activity.

For us is more important the technology we are using and testing than the project itself and features. This project has a didactic purpose, it is designed fo learning Sage-Code tech stack. We may change the technology in the future.

Tech Stack

We are thinking to use Go language for this project. We have already chose PostgreSQL database and will probably use HTMLX or Svelte. We do not know yet what exactly will be used for interacting to PostgreSQL database.

Core Team
